Design System & Conventions

Colors

  • #0f172a — dark background
  • #64748b — muted text
  • #e2e8f0 — primary text
  • green — success / delivered
  • blue — info / processing
  • yellow — warning / low stock
  • red — danger / backordered

Patterns

  • Tailwind CSS utility classes (preferred over custom CSS)
  • Custom SVG for all charts (no chart library)
  • CSS Grid for layouts
  • Modals via Vue Teleport to body
  • No emojis in UI
  • v-for uses semantic key (sku, id), never index

Data Conventions

  • Dates: ISO 8601 YYYY-MM-DD
  • Month filter: YYYY-MM or Q#-YYYY
  • Filter default: all (no filter)
  • Revenue goal: $800K/month, $9.6M YTD
  • Validate dates before .getMonth()
  • Inventory has no month/time dimension
